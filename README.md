# Auto.js 脚本

## [微信僵尸好友脚本](./we_chat_zombie_friends/)
> 查找被对方删除、拉黑的微信好友
>
> 通过转账前的提示信息判断好友关系

### 非正常好友转账提示信息
1. 被拉黑：请确认你和他（她）的好友关系是否正常
2. 被删除：你不是收款方好友，对方添加你为好友后才能发起转账

### 支持的语言和微信版本
* 适用微信7.0.9~7.0.16版本
* 仅支持简体中文
